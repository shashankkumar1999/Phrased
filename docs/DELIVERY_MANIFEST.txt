BUG FIXES IN v1.0.0
================================================================================

1. Claude Variable Input Focus Jumping
   - Issue: Focus jumps from sidebar to Claude chat
   - Status: ✅ FIXED
   - Solution: Added event propagation blocking and focus trap
   - Location: src/contents/sidebar.tsx (handleInputKeyDown function)

2. ChatGPT Extension Loading/Disappearing
   - Issue: Extension doesn't load or disappears on ChatGPT
   - Status: ✅ IMPROVED
   - Solution: Multiple selector fallbacks, better error handling
   - Location: src/lib/inject.ts (selector strategy)

3. Perplexity Injection Not Working
   - Issue: Inject fails, only copy works
   - Status: ✅ IMPROVED
   - Solution: Visible textarea detection, additional event dispatch
   - Location: src/lib/inject.ts (Perplexity case)

INSTALLATION INSTRUCTIONS
================================================================================

1. Download: build/chrome-mv3-prod.zip
2. Extract: Unzip to any folder (e.g., ~/phrased-extension/)
3. Open: chrome://extensions
4. Enable: Developer mode (top-right toggle)
5. Load: Click "Load unpacked"
6. Select: The extracted chrome-mv3-prod folder
7. Done: Orange "Phrased" tab appears on any website
