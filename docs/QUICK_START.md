# Phrased — Quick Start (30 seconds)

## Installation

1. Download `phrased-extension.zip` from releases
2. Extract it: `build/chrome-mv3-prod/`
3. Open `chrome://extensions`
4. Enable "Developer mode" (top-right toggle)
5. Click "Load unpacked"
6. Select the extracted folder
7. Done! You'll see the orange **Phrased** tab on any website

## Basic Usage

1. **Click the orange "Phrased" tab** on the right edge
2. **Search or browse** the 100 prompts
3. **Fill in the variables** (blue input boxes)
4. **Preview** shows your prompt with filled values highlighted orange
5. **Click "Copy prompt"** to copy, or **"Send to chat"** to inject
6. **Paste or use** the prompt in your AI chat

## Keyboard Shortcuts

| Key | Action |
|-----|--------|

## Tips

💡 **Fuzzy search** works great — try typing just a few letters
🔍 **Filter by category** — click the pill buttons at the top
📋 **Live preview** — see your substitutions instantly highlighted
⚡ **Copy always works** — if inject fails, copy and paste manually
🎯 **Green dot** = variable filled, ready to go

## Troubleshooting

**Extension not showing?**
- Reload page (`Ctrl+R` or `Cmd+R`)
- Check `chrome://extensions` — enabled?

**Focus jumping to chat on Claude?**
- Click back in the Phrased input
- Use `Tab` to move between variables

**Inject not working?**
- Use **Copy** instead (works on any site)
- Reload the page and try again
- Check if your platform is supported (see table above)

**Can't find a prompt?**
- Try different search terms
- Filter by category
- Browse "All" to see everything
