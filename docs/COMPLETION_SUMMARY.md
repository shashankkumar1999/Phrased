## 📦 Installation (2 Minutes)

```bash
1. Download: build/chrome-mv3-prod.zip
2. Extract to any folder
3. Open: chrome://extensions
4. Enable: Developer mode (top-right toggle)
5. Click: "Load unpacked"
6. Select: The extracted folder
7. Done! Click the orange "Phrased" tab to start
```

## 🚀 New Feature: Try Inject

**For users on unsupported AI platforms:**

1. Open Phrased on any unsupported AI (Canva, custom tools, etc.)
2. Click **"Try Inject"** button (appears in category pills)
3. Button turns orange, "Send to chat" appears
4. Select a prompt, fill variables, click "Send to chat"
5. If it works: prompt injected into chat ✅
6. If it doesn't work: use Copy button, paste manually ✅

**Works with any website that has:**
- contenteditable input
- textarea element
- text input field

**Fallback**: Copy always works, paste manually
